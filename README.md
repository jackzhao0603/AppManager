# How to use
## Quick start
1. Add it in your root build.gradle at the end of repositories:
    ```gradle
    allprojects {
        repositories {
            ...
            maven { url "https://jitpack.io" }
        }
    }
    ```
2. Add the dependency
    ```gradle
     dependencies {
	       implementation 'com.github.jackzhao0603:AppManager:v0.1.1'
	 }

    ```
